# HK_TitleScreenRemover
HK_TitleScreenRemover is a plugin for RPG Maker MV that completely removes the default title screen. This allows users to create their own title screens from events, or if a title screen just isn't suitable for their game.

Reference this video for a full tutorial on how to use this plugin.
